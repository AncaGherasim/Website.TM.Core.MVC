Known Traveler Number
The Known Traveler Number (TSA PreCheck program, Global Entry, NEXUS, SENTRI) is a Trusted Traveler. This is a Membership ID/Pass ID that is issued by the government to identify passengers that participate in the programs mentioned.

Redress Number
The Homeland Security(DHS) provides a unique identification number issued by the Transportation Security Administration(TSA) for passengers whose names have experienced secondary security Screening. These passenger names are similar to those on the current terrorist watch list. If you have a Redress number you must provide it.
